      关于 CSS 学习总结

      # 选择器

      选择指定元素设置样式，格式为

      选择器 {
    css属性: 值;
}

      # 普通选择器

      1、通配符选择器(*) 用来选择所有元素，包括 body、html；也可以用来选择某个元素下的所有元素；

      * {
	font-size: 14px;
}

#testWrap * {
	font-size: 16px;
}

      出于性能考虑不建议使用，通配符会遍历并命中文档中的所有元素

      2、HTML 标签选择器，为某种类型的标签设置样式

      a {
	text-decoration: none;
}

      3、id 选择器，根据唯一 id 标识来选择元素

      #testWrap {
	font-size: 16px;
}

      HTML 和 XHTML 对类和 ID 值定义都是严格区分大小的

      4、类名选择器,可以多个类名组合，也可以和一些选择组合使用

      .test {
	font-size: 14px;
}

.test.active {
	font-size: 16px;
}

p.test {
	font-size: 18px;
}

      5、包含选择器，选择某个元素的子孙元素

      .test p {
	font-size: 14px;
}

      6、子元素选择器（>），只能命中子元素，不能命中孙辈等往下元素

      .test > p {
	font-size: 14px;
}

      7、相邻选择器（+），选择某个元素之后紧跟的第一个元素，即第一个兄弟元素

      /* 选择test类名后的紧跟的第一个元素，如果是p标签，则设置红色，不是p则无法选中 */
.test + p {
	color: red;
}

      8、兄弟选择器(~)，选择某个元素后面的所有被命中的兄弟元素,不满足则无法命中

      .test > p {
	font-size: 14px;
}

      # 属性选择器

      根据元素的属性及属性值选择

      /* 1、同时具有href和title属性，只要具有该属性即可，即使属性值为空 */
a[href][title] {
    color: red;
}

/* 2、属性值等于选择指定属性值，必须完全匹配；如果属性有空格隔开的多个，则无法匹配，如class */
input[type="text"] {
    color: #333;
}

/* 3、多个属性值有一个值完全等于指定属性值；如class为is test，可以匹配到is；但如果class为is-active test则无法匹配 */
div[class~="is"] {
    color: #333;
}

/* 4、多个属性值以指定属性值开头；如class为is test，可以匹配到以is开头； */
div[class^="is"] {
    color: #333;
}

/* 5、多个属性值以指定属性值结尾；如class为is test，可以匹配到以test结尾； */
div[class$="test"] {
    color: #333;
}

/* 6、任意属性值包含指定字符串；如class为is-active test，可以匹配到包含is； */
div[class*="is"] {
    color: #333;
}

/* 7、匹配属性值为value开头或value-开头；如class为en-us test或en均可被匹配，但test en或c-en不能匹配 */
div[class|="en"] {
    color: #333;
}

      # 链接伪类选择器

      /* 1、设置未访问过的超链接样式 */
a:link {
    color: yellow;
}

/* 2、设置已访问过的超链接样式 */
a:visited {
    color: yellow;
}

/* 3、设置超链接鼠标指向样式 */
a:hover {
    color: yellow;
}

/* 4、设置超链接被访问时样式（按键按下） */
a:active {
    color: yellow;
}

      # dom结构伪类选择器

      1、first-child选择第一个子元素

      2、last-child选择最后一个子元素

      3、nth-child(an+b)选择符合规则个数的子元素；可以通过odd代表奇数，event代表偶数；同时不能写成b+an形式

      4、nth-last-child(an+b)，同上倒数

      特别注意，上述选择器必须要满足指定位置且条件符合。如.test p:first-child，当.test下第一个元素不是p时，无法匹配

      /* 1、匹配lang=指定值的元素 */
p:lang(it) {
    color: red;
}

/* 2、不符合指定条件的元素，条件不能是伪类选择器。可以是普通选择器和属性选择器等 */
div:not([class^="isActive"]) {
    color: red;
}

/* 3、root匹配文档树的根元素，html */
:root {
    font-size: 16px;
}

/* 4、first-child选择第一个子元素，注意和其他选择器组合时要同时满足；当第一个元素不是p时无法满足条件 */
.test p:first-child {
    font-size: 16px;
}

/* 5、同理，last-child选择最后一个子元素 */

      5、first-of-type、last-of-type、nth-of-type、nth-last-of-type与first-child类似，但两者的区别在于first-of-type是选择 第一个元素E，即使E不是第一个元素

      <style>
    /* 即使div不是第一个节点，可以选中2222变为红色 */
    .type-wrap div:first-of-type {
        color: red;
    }
    /* 即div不是第一个节点，不匹配 */
    .type-wrap div:first-child {
        color: red;
    }
</style>
<body>
        <div class="type-wrap">
            <p>1111</p>
            <div>2222</div>
            <div>3333</div>
            <div>4444</div>
        </div>
</body>

      6、:empty匹配没有子元素的节点，注释或处理指令不影响

      7、:target用于加亮当前链接锚点的部分；如地址为xxx#section，那么:target就指向了id为section的元素

      # UI状态伪类选择器

      用于选择元素处于某种UI状态而设定样式

      /* 1、focus，该元素获得焦点，被点击，如div、input输入等；focus-within则是获得焦点是元素的子孙节点时匹配，如div的下的input获得焦点，会匹配该div，给div添加红色边框 */
input:focus {
    color: red;
}
div:focus-within {
    border: 1px solid red;
}

/* 2、checked，元素被选中，如select的option，checkbox等 */
option:checked {
    color: red;
}

/* 3、enabled，元素可用时；相对的有disabled选择元素禁用时 */
input:enabled {
    color: red; 
}

/* 4、read-only，不可编辑只读状态，如文本输入框，火狐并不支持；相对的有read-write代表可以编辑 */
input:read-only {
  background-color: #ccc;
}

/* 5、optional，没有require属性的元素；相对的required则是选择具有required的必填元素 */

/* 6、in-range，当前值处于min、max限定范围的元素，相对的out-of-range则是当前值不在min、max之间的元素 */

/* 7、defalut，标记默认状态（单选按钮，checkbox，select的默认选中），可以用来做推荐标记,如多种单选支付方式，支付宝，微信，给选中的项加上推荐 */
input:default + label::after {
    content: '（推荐）';
}

      # 元素内容选择器

      ::双冒号是css3新语法，如果需要兼容ie8-，可以使用css2语法:单冒号

      1、::first-letter、::first-line

      /* 1、设置第一个字符样式，作用于块级元素，且只能运用少部分css：字体、背景、边距、边框 */
p::first-letter {
    color: red;
}

/* 2、设置第一行字符样式，也是作用于块级元素，且只能运用于少部分css：字体、 背景*/
p::first-line {
    color: red;
}

      2、::before、::after结合content属性为元素前后添加修饰性内容，可以使用attr获取元素属性值

      .must::before {
    content: attr(title)
}

      3、::placeholder设置占位文本样式

      4、::selection设置鼠标选中内容样式，常用来做高亮

      # 文本样式

      1、text-transform 文本大小写控制

      capitalize: 每个单词的首字母转化为大写
uppercase：全部转为大写
lowercase：全部转为小写

      2、white-space设置元素中的空格符

      normal: 默认方式，连续的空白符会合并为一个，换行符会被处理为空白符
pre: 连续的空白符会被保留。在遇到换行符或者<br>元素时才会换行，当文字超出边界时不换行
nowrap：强制在同一行内显示所有文本，合并空白符，直到文本结束或者遭遇br对象。连续的空白符会被合并。但文本内的换行无效
pre-wrap: 当文字碰到边界时发生换行
pre-line: 保持文本的换行，不保留文字间的空白距离，当文字碰到边界时发生换行
break-spaces: 与 pre-wrap的行为相同
      属性 换行符号 空格 边界换行 行尾空格
      normal 合并 合并 换行 删除
      pre 合并 合并 不换行 删除
      nowrap 保留 保留 不换行 保留
      pre-wrap 保留 保留 换行 挂起
      pre-line 保留 合并 换行 删除
      break-spaces 保留 保留 换行 换行

      3、word-break设置换行行为；与word-break相比，overflow-wrap仅在无法将整个单词放在自己的行而不会溢出的情况下才会产生中断

      normal: 允许单词中间换行
break-all：同normal
keep-all：保持单词的连续

      4、word-wrap内容超过指定容器的边界时是否断行

      normal: 允许内容溢出
break-word：超出换行，允许单词中间换行

      5、word-spacing设置单词之间的距离；letter-spacing则是设置字符之间的距离

      6、line-height设置行高，对于一行元素line-height设置为height可以实现垂直居中

      7、text-decoration设置修饰线

      text-decoration-line:指定文本装饰的种类。相当于CSS2.1的text-decoration属性，可取值:none | underline | overline | line-through | blink
text-decoration-style:指定文本装饰的样式。如波浪线wavy、实线solid、虚线dashed
text-decoration-color:指定文本装饰的颜色。

      8、text-shadow设置文字阴影

      # 字体

      1、font 复合字体属性

      2、font-family 字体系列

      3、font-size 字号大小

      4、font-style 字体样式

      normal：标准字，默认
oblique: 倾斜字体
italic：斜体字

      5、font-variant 设置字体是否为大写小型字

      normal: 标准字
small-caps：英文将显示为小型的大写字母

      6、font-weight 字体粗细 既可以使用 500、600、700 这种，也可以使用 bold（加粗）、normal、bolder（更粗）、lighter（更细）；但在小程序下使用 600、700 加粗会失效，因此使用 bold 等加粗

      7、@font-face 设置自定义或网上下载的字体 ttf、eot 等格式

      # 动画

      1、@keyframes 定义一个动画，定义的动画名称用来给 animation-name 使用

      2、animation 复合属性，应用@keyframes 设置的动画

      3、animation-name，应用的动画名，必须由@keyframes 定义

      4、animation-duration，动画一个周期持续时间，默认为 0 不动画，可以设置秒或毫秒如 5s 或 5000ms

      5、animation-timing-function，动画过渡类型

      linear: 线性，动画从开始到结束具有相同速度
ease：缓慢的开始，然后快，然后在慢结束
ease-in: 缓慢开始
ease-out: 缓慢结束
ease-in-out：缓慢开始和缓慢结束
cubic-bezier(n,n,n,n)：立方贝塞尔函数定义的速度曲线

      6、animation-delay 动画延迟时间

      7、animation-iteration-count，动画循环次数，默认为 1，infinite 代表无限循环

      8、animation-direction，动画播放是否反向

      normal: 正常播放
reverse： 反向播放
alternate：奇数次时正向，偶数次数时反向
alternate-reverse：奇数次时反向，偶数次时正向

      效果可以参考 50%分割开的往返动画，完全可以利用方向来实现。这个属性非常有用，当有动画需要往返时，可以考虑到该属性

      代码示例:

      /* 两种动画等价，一种利用animation-direction实现（推荐），一种50%明显代码量更多 */
.move-wrap {
	position: relative;
	width: 100px;
	height: 100px;
	background-color: blueviolet;
	animation: move 4s infinite;
	/* animation: move2 2s infinite alternate; */
}

@keyframes move {
	0% {
		top: 0;
	}

	50% {
		top: 200px;
	}

	100% {
		top: 0;
	}
}

@keyframes move2 {
	0% {
		top: 0;
	}

	100% {
		top: 200px;
	}
}

      # 背景

      1、background 复合属性，设置背景特性

      2、background-attachment 背景图是否跟随内容滚动，必须先设置 background-image

      scroll: 跟随页面其他内容滚动，默认
fixed: 背景固定
local: 相对于元素自身内容定位，开始固定，跟随滚动元素滚动

      对于 scroll，背景图跟随内容滚动而滚动，但如果内容本身可以滚动，背景是不会跟随滚动的，而 local 则是会跟随滚动的

      3、background-color 元素背景色

      4、background-image 元素背景图，可以有多张

      5、background-position: 50px 100px;背景图位置，水平与垂直方向，除了 left、center、right 和 top、center、bottom 的组合还可以采用像素来控制，必须先设置 background-image

      6、background-size: 80px, 80px;背景图尺寸，常用来在为了高清时，图片二倍图，设置 size 为 1 倍可以获得更清晰的效果

      7、background-repeat，背景重复

      repeat：在水平和垂直方向重复，默认
no-repeat：不重复
repeat-x：x方向重复
repeat-y：y方向重复

      背景重复可以让我们使用一个极小的背景图来铺满一个极大的空间

      8、background-origin，背景图相对位置，如果 background-attachment 为 fixed 则相对位置不会起作用

      padding-box：相对填充框
border-box：相对边界
ontent-box：相对内容框

      9、background-clip 背景图绘制区域，同上

      <style>
	div {

		overflow: scroll;

	}
</style>

      # 边框与轮廓

      1、border 复合边框属性

      2、border-color 边框颜色

      3、border-style 边框样式

      none: 无边框
hidden：与none相同，但对于表格，hidden用以解决边框冲突
dotted：点状边框
dashed：虚线边框
solid：实线边框
double：双线边框

      虚线和点状边框在大多数浏览器下呈现实线状

      4、border-width 边框宽度

      5、border-bottom 复合下边框属性

      6、border-bottom-color 下边框颜色

      7、border-bottom-style 下边框样式

      8、border-bottom-width 下边框宽度

      9、类似 bottom 还有 top、left、right 控制各方向的边框

      10、outline，外部轮廓，与 border 类似具有 outline-style（值同 border）、outline-width、outline-color 三个属性

      11、border-radius，边框圆角，可以单独控制四个方向的圆角值不同来绘制各种图形，常用的值 50%可以用来绘制圆形

      12、border-image，使用边框图片来替换 border-style，但更多的是使用 background 来实现这种效果；参考链接: http://c.biancheng.net/css3/border-image.html

      13、box-shaow 为方框添加一个或多个阴影。box-shadow: 10px 10px 5px #888888;

      可以利用 border 来三角形，圆形，阴影等很多有趣的形状

      # 溢出与透明

      1、overflow-x 横向溢出效果

      visible：不裁剪内容，显示在边框之外
hidden: 裁剪内容，不滚动
scroll：裁剪内容，滚动机制
auto：如果溢出则滚动
no-display：如果内容框溢出，则删除内容框
no-content：如果内容框溢出，则隐藏整个内容

      2、overflow-y 纵向溢出效果，取值同上

      3、overflow 横向与纵向

      4、opacity 透明度，0 为完全透明，1 为不带任何透明度

      # 边距与尺寸

      1、width 元素宽度

      2、height 元素高度

      3、max-width、min-wdith 最大，最小宽度

      4、max-height、min-height 最大，最小高度

      5、margin，向外延伸，具有 margin-top、margin-bottom、margin-left、margin-right 四个方向

      6、padding，向内填充，同样具有 padding-top、padding-bottom、padding-left、padding-right

      注意：非 inline 元素的边距、width 等都不会生效

      # Flex 布局

      1、flex 复合属性空间分配，flex-grow、flex-shrink、flex-basis

      2、flex-grow 弹性盒子扩展比例

      3、flex-shrink 弹性盒子压缩比例

      4、flex-basis 伸缩基准值

      5、flex-direction，主轴方向

      row：横向
row-reverse：横向反方向
column：纵向
column-reverse：纵向反方向

      6、flex-wrap 控制换行

      nowrap: 不换行
wrap：必要时自动换行，自左向右排
wrap-reverse：换行后反向排列，s型排列

      7、align-content 纵向对齐

      stretch: 容器开头无间距，中间、最后平分间距
center：项目居中
flex-start：项目位于容器开头
flex-end：项目位于容器结尾
space-between：项目之间平分间距，最后没有
space-around：项目开头、之间、结尾间距相等

      8、align-items 设置 flex 容器的纵向对齐

      stretch: 拉伸铺满
center：居于容器中间
flex-start：居于容器开头
flex-end：居于容器结尾
baseline：容器基线，基线不调整时同flex-start

      9、justify-content 设置 flex 容器的水平对齐】

      1、content，配合::before 与::after 伪类来插入内容 content 可以获取元素的任意 attr 属性，这一点可以用来实现类似 title、tooltips 等效果

      <!DOCTYPE html>
<html>
	<head>
		<style>
			a::after {
				content: attr(href);
			}
		</style>
	</head>

	<body>
		<p><a href="http://www.w3cschool.cn">w3cschool</a> 包含免费的教程和参考.</p>
	</body>
</html>

      2、counter-reset 创建或重置计数器

      3、counter-increment 递增或递减计数器

      两者结合可以用来实现自动章节计算等自动计数功能

      <!DOCTYPE html>
<html>
	<head>
		<style>
			body {
				counter-reset: section;
			}
			h1 {
				counter-reset: subsection;
			}
			h1:before {
				counter-increment: section 1;
				content: 'Section ' counter(section) '. ';
			}
			h2:before {
				counter-increment: subsection 1;
				content: counter(section) '.' counter(subsection) ' ';
			}
		</style>
	</head>

	<body>
		<p><b>请注意:</b> IE8仅在指定了!DOCTYPE的情况下才支持这些属性.</p>

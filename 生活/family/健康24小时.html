            # 7.30轻轻的唤醒身心

            # 慢慢醒来

            清晨醒来，赖一会床（5-10分钟），慢慢醒来

            1、伸伸懒腰

            2、转动眼睛

            3、叩齿运动

            4、按摩肚脐

            5、伸曲双腿

            6、左右翻身

            # 刷牙

            温水刷牙、倾斜45度刷、刷牙3个3（早中晚3次，每次3分钟，每3个月更换牙刷）

            # 毛巾清洗

            白醋与水混合清洗、2勺盐化入水中 均为浸泡后清洗；每周清洗1-2次；同时3个月更换一次

            # 早起一杯水

            20度左右温水或蜂蜜水，100ml就足够了，不超过200ml；刷牙后空腹喝，慢慢喝

            # 早餐

            早餐搭配：牛奶豆浆+面包馒头+鸡蛋+蔬菜水果

            牛奶，豆浆不能空腹吃，搭配面包、馒头等；每天吃豆三钱，何须服药连连

            鸡蛋煮8分钟，煮时放少许盐，不要用冷水泡（蛋壳已经没有防细菌能力）

            剩饭剩菜尽量不要吃，会产生致癌的亚硝酸盐；避免油炸食品

            # 豆浆

            玉米小米豆浆：50克黄豆，小米玉米碎15克，白糖适量；谷类与豆类营养搭配

            红枣莲子豆浆：50克黄豆，红枣去核，莲子15克，白糖适量；红枣、莲子富含维生素及矿物质，与黄豆搭配制作豆浆，有很好的健脑益智、延缓衰老、降低胆固醇、稳定情绪等功效

            核桃花生豆浆：黄豆50克，花生仁、核桃仁各15克，白糖适量；花生、核桃都是搭配黄豆制作豆浆的好食材，有良好的抗衰老、美容颜、增强记忆的作用。

            # 早餐粥

            红糖小米粥：小米150克，红糖10克（粥煮好加即可）；健脾暖胃、益气补血

            红豆紫米粥：红豆50克，紫米100克（加适量糖）；补血养肝、滋阴强肾。

            板栗红枣粥：熟板栗、红枣各10个，大米150克（可加糖）；暖身，还可健脾胃、养肝肾

            山药南瓜粥：山药、南瓜各50克，大米100克（加适量盐）；补气润肺、镇咳祛痰、健脾和胃、通便排毒的功效

            花生双豆粥：花生50克，绿豆、黑豆各30克（加适量糖），大米100克；养血、强身

            香蕉双米粥：香蕉（或苹果）200克，大米、糯米各50克；促进食欲、通便排毒

            青菜瘦肉粥：青菜，瘦肉；补充蛋白质，增强免疫力

            # 注意事项

            避免零食早餐，走食早餐

            牛奶不能高温煮沸，70度最佳；煮牛奶避免加糖等；和牛奶避免使用酸性水果（果酸影响牛奶吸收，一小时后再吃）

            豆浆保温会大量繁殖细菌；豆浆不宜超过300ml；豆浆中不能放红糖

            # 8.30拥抱阳光

            不要错过8-9点的阳光；以30分钟为宜；避免11点-13点晒太阳

            上班路上巧锻炼：头部微向前倾，双手交叉放于脑后向前发力，同时颈部向后发力；踮踮脚尖

            晨练后不要睡回笼觉

            # 9.30工作不可过劳

            整理办公桌：工作效率的高低往往与工作环境有着很大关系，杂乱无章的工作环境往往会令工作效率低下，文件归类，垃圾清场，整洁的环境会让你的心情大好，而简单的劳动也能让身心得以放松。

            装饰：摆放一些绿植，花点时间装饰一下自己的工作环境，例如，挂一幅孩子的画，放些心灵成长的书籍

            坐姿：避免身体前倾；不要跷二郎腿。不仅会造成腰酸痛，而且会对生殖系统产生不利影响

            喝水：每隔一小时喝水一杯，伸伸懒腰

            呼吸操：每天10分钟激励呼吸操，缓缓吸气，使腹部鼓起，然后再把气慢慢呼出。动作从容舒适，有安定神经的作用。

            # 12.00午后时间

            午餐推荐的几种食物：西蓝花（抗衰老、增强免疫力）、圆白菜（有益肠胃、保持活力）、胡萝卜（保护视力、延缓衰老）、豆腐（促进消化、提高记忆力）、鱼肉（补充蛋白质、保护心血管）、牛肉（增长体力、补充元气）

            凉拌西蓝花、胡萝卜丝炒豆芽、西蓝花炒牛肉、虾仁蒸豆腐、红烧鱼、葱爆鳝鱼片、香菇草鱼块、土豆闷牛肉、白菜豆腐汤

            午餐，晚餐正餐饭后不宜立即吃水果，最好等待1-2小时后再吃水果

            午睡：小憩片刻30分钟，不宜太久，陷入深度睡眠反而更易疲惫；超过3点后不宜再午睡

            午睡后恢复活力：轻轻拍打前额；起来慢慢走动一会；洗一把脸；整理桌位

            颈部下垂：站在办公桌旁，弯腰，头部往下，与双手自然垂放向地面，维持20秒。该伸展操可舒缓脖子疲劳感，帮助血液循环。注意：此过程中不需要用力，保持身体一定的柔软。

            双臂上举：坐着，背脊挺直，深吸一口气，双臂敞开并向上伸展。接着呼气时慢慢将身体向后弯曲，此时手臂维持不动。保持这个姿势几秒钟后，将手臂慢慢放下，滑过身体两侧。可以舒缓久坐不动造成的背部酸痛。

            手腕运动：将手掌心朝外，并用另一只手压住指尖作辅助，再将手掌往内扳。停留约10秒后，掌心朝内（自己），并将手掌往外扳，重复几次。可以舒缓因长时间打字所造成的前臂与手腕酸痛

            转动腰部：坐在椅子上背脊挺直，下半身不动，上半身往左边转，停留几秒并同时调节呼吸，再往另一侧转。可以舒缓腰部、活动筋骨，还可帮助消化

            多仰仰头：如果在室内不方便活动，那上班族可以在工作之余多仰仰头。养成经常后仰头颈的习惯，能帮助消除头颈肌肉紧张。

            下午茶：菊花茶、枸杞子茶、玫瑰花茶、绿茶、大麦茶

            # 17.30放下工作，把快乐带回家

            列个清单：在下班前两个小时，简单列个清单，弄清楚哪些是你今天必须完成的工作、哪些工作可以留待明天，这样你就有充足的时间来完成任务，而且分清轻重缓急也是做好工作的前提。

            一天中运动的黄金时间，经过一天的脑力劳动后，下班后适当地进行锻炼，可以缓解一天的压力。自然界中绿色植物进行一天的光合作用后，积累的氧气含量在这个时候也达到了最大值

            一方面进行跑步、跳舞、打球等体力活动，另一方面可以进行读书、写作、下棋等思维锻炼

            跳绳是一种简单有效的运动方式；跳绳前须热身；每次跳绳的时间不宜过长，以30分钟至1小时为宜，每周跳绳不应少于4次，但也不可多于6次

            晚饭后散步：散步的时间应在饭后半小时至一小时。如果胃中的食物还没完全消化掉就散步，易诱发功能性消化不良，而且食物存留在胃中，久而久之还易引起胃下垂。

            聊天谈心，脱离手机

            # 20.00练习瑜伽释放压力

            冥想、打坐、扎马步

            刷牙后不要再进食，进食应漱口

            尽量在11点前入睡

            咬牙切齿：空口咬牙，反复咬紧放松

            新冠感染后温补气血：山药、秋葵、狗肉、羊肉、泥鳅

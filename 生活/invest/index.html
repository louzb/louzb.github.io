# 投资理财

# 一些概念

# PE

市盈率（PE）是股票价值与每股盈利的比值，是用来衡量股票价格与价值的比例指标，PE越高说明价格与价值 的背离度就越高，市盈率越低，其股票越具有投资价值

# PB

市帐率（PB），市净率越低，投资风险小。所以在采用市净率指标用于投资指数选择时，通常是选择那些市净率较低的股票或者市净率较低的行业指数。 PB相对较低，其投资价值相对更高。从表一我们可以看出，目前标普A股红利指数，相应的市净率要低于主流宽基指数如沪深300、中证500等

分析过PB、PE等指标后，投资者需要注意的是，在投资指数基金时，某个指数代表了一篮子基股票， 可能来自不同行业，也可能是同一行业，这时应该纵向比较PE、PB。 当目前的PE、PB相比历史数据较低时，说明该指数当前估值被低估，有配置价值。

# 指数基金怎么选

1、利用指数红绿灯，原则上选择被低估的指数

2、利用基金搜索（天天基金），搜索指数

3、选择基金主要关注：（基金成立日期3-5年； 基金规模最好大于3亿； 基金公司选择优秀的公司； 跟踪误差小，0.1左右就很小了； 买入费低）

4、投资理念

能力圈-你要足够了解你所投资的股票，清楚他的价值是多少，不懂的不投

安全边际-你要投资的股票在价格远远低于它的价值时开始投资

                          句子迷

                          # 知也无涯

                          业精于勤，荒于嬉，行成于思，毁于随

                          吾生也有涯，而知也无涯

                          黄金时代是在我们的前面，而不在我们的后面

                          学问二字，须要拆开看，学是学，问是问

                          人生有一道难题，那就是如何使一寸光阴等于一寸生命

                          追踪着鹿的猎人是看不见山的

                          死犹未肯输心去，贫亦其能奈我何！

                          才自清明志自高

                          志当存高远

                          贤者不悲其身之死，而忧其国之衰

                          # 白驹过隙

                          科学是没有国界的，因为它是属于全人类的财富，是照亮世界的火把，但是学者是有国界的，他的荣耀，永远属于他的祖国

                          人生天地之间，若白驹之过隙，忽然而已

                          夫子温、良、恭、俭、让

                          醉斩长鲸倚天剑，笑凌骇浪济川舟

                          勿以朱颜好，而忘白发生

                          劝君莫惜金缕衣，劝君须惜少年时。有花堪折直须折，莫待无花空叹息

                          雄关漫道真如铁，而今迈步从头越

                          三餐四食混温饱"，下联"五枝六叶度余生

                          记得当时月，回首不堪怜

                          妾居昆仑山，君住东海上，相隔万里遥，咫尺一梦长，游鱼传尺素，春水寄相思，一掬多少泪，问君知不知

                          # 不遭人妒是庸才

                          问春风，相思何物？海角天涯，千丝万缕，全是癫狂柳絮。万水千山又一年，檐前归燕，知否， 伊人消息人道离恨如春草，更行更远还生。偏又逢，梅子黄时雨，怎奈得，这次第，只恨此身非鱼， 一江春水，绵绵东流去

                          平生不惧鬼神怒，不遭天妒是庸才

                          时穷节乃现，一一垂丹青

                          你会爱我多久？一直到你不爱我为止。

                          钱钟书：从此以后，只有死别，没有生离

                          自觉此心无一事，小鱼跳出绿萍中

                          君何独久坐，伏夜意难平

                          娄山关：雄关漫道真如铁，而今迈过从头越

                          中堂何曾负天下，天下史书尽负他

                          请君试问东流水，日夜东流何时歇

                          # 伴君朝与暮

                          感君一回顾，伴君朝与暮

                          何不策高足

                          独坐水亭风满袖，世间清景最微凉

                          曾相思许诺，曾遗憾错过，桃花飘零

                          青冥辞去张居正，四海勤思留考成

                          贾不假，白玉为堂金做马

                          抱膝思量，莫道不凄凉。早近持觞，暗思何事断人肠，曾是向他人梦里

                          从今莫把仇人靠，千种相思一撇销

                          别回头,身后万家灯火,都不是你的归宿

                          天可补，海可填，南山可移。日月既往，不可复追。- 曾国藩

                          # 生前何必久睡

                          生前何必久睡，死后自会长眠。

                          由爱而生的勇气，足以抵抗世间所有的孤独。

                          但愿日子清净抬头遇见的满是柔情

                          最长的路也有尽头，最黑暗的夜晚也会迎接清晨。

                          于她，虽千万人吾往矣

                          你的孤独，虽败犹荣

                          我与春风皆过客，你携秋水揽星河

                          三生有幸遇见你，纵然悲凉也是情。

                          愿有岁月可回首，且以深情共白头。

                          既许一人以偏爱，愿尽余生之慷慨。

                          # 我愿提笔画尽天下

                          渐行渐远渐无书，水阔鱼沉何处问。

                          半身风雨半身伤，半句别恨半心凉。

                          只缘感君一回顾，使我思君朝与暮。

                          此情无计可消除，才下眉头，却上心头。

                          既不回头，何必不忘。

                          以你之名，冠我之姓。-我们结婚吧

                          我愿提笔画尽天下，许你一世繁华。

                          海底月是天上月，眼前人是心上人。

                          风追落叶叶追尘，彩云追月月自明。-落花有情流水无意

                          日日思君不见君

                          # 十里桃花竟折枝

                          此生固短，无你何欢

                          山有木兮木有枝，心悦君兮君不知

                          我本将心向明月 奈何明月照沟渠

                          回身笑看三生断，十里桃花竟折枝

                          与君同舟渡，达岸各自归

                          仰不愧于天，俯不怍于人

                          有时爱情徒有虚名

                          有时一本正经讲大道理的傻样，也能让人痴迷了好久

                          岁岁年年人不同,年年岁岁花相似

                          一愿社稷昌，二愿黎民安，三愿我所爱无忧无恙，岁岁长安

                          # 梦想的破坏者

                          未来的我值得更好的生活，所以现在的我必须努力

                          任何值得去的地方，都没有捷径

                          梦想的破坏者-哥本华金句

                          生命是一团欲望，欲望不能满足便痛苦，满足便无聊，人生就在痛苦和无聊之间摇摆。

                          只有幸福的人，才会把无关痛痒的小事挂在心上。

                          如果我们的存在是否具有价值需要依靠别人来评定，我们的生命就是可悲的。

                          人们给同类施加痛苦并无其他原因，仅仅是出于恶意。在所有的动物中，唯有人这么做。

                          人就像寒冬里的刺猬，互相靠得太近，会觉得刺痛；彼此离得太远，却又会感觉寒冷；人是必须保持适当的距离过活。

                          我们的不幸往往由于我们对于幸福的追求！我们做事之初喜欢抱着一种信念：我们一定能在世间找到某种幸福。

                          对人来说，金钱是一种抽象的幸福，那些不能再具体享受人生幸福的人，便只好费尽心机去谋取钱财。

                          人类所能犯的最大的错误就是拿健康来换取其他身外之物！

                          # 庸人自扰之

                          家中何事？松花酿酒，春水煎茶。

                          天下本无事，庸人自扰之！所谓杞人无事忧天倾是也

                          笑，全世界便与你同声笑；哭，你便独自哭。

                          你如果认识从前的我，也许你会原谅现在的我。

                          对于大多数的女人，"爱"的意思就是"被爱"。

                          无条件的爱是可钦佩的——唯一的危险就是：迟早理想要撞着了现实，每每使他们倒抽一口凉气，把心渐渐冷了。

                          于千万人之中遇见你所遇见的人，于千万年之中，时间的无涯的荒野里，没有早一步，也没有晚一步，刚巧赶上了，那也没有别的话可说，惟有轻轻的问一声："噢，你也在这里吗？"

                          年轻人的誓言不是说说而已。对于三十岁以后的人来说，十年八年不过是指缝间的事； 而对于年轻人而言，三年五年就可以是一生一世。

                          你问我爱你值不值得，其实你应该知道，爱就是不问值得不值得

                          但是，酒在肚子里，事在心里，中间总好象隔着一层，无论喝多少酒，都淹不到心上去。

                          # 何不策高足

                          纸上得来终觉浅，绝知此事要躬行。

                          挽弓当挽强，用箭当用长。射人先射马，擒贼先擒王。

                          何不策高足，先据要路津

                          古今多少事，都付笑谈中。

                          你是我眼中百看不厌的星辰大海

                          余生漫漫，柔情至死

                          多少相思，多少离愁，终成一道水痕送我远走。

                          我用一千次回眸，换得今生在你面前的驻足停留。

                          爱任何事物的方法，就是要意识到：你可能会失去它。

                          乌合之众：人一到群体中，智商就严重降低，为了获得认同，个体愿意抛弃是非，用智商去换取那份让人备感安全的归属感。

                          # 闲愁奈我何

                          人生真的没有意义。但人生也好在无意义，才容得下各自赋予意义。

                          你站在桥上看风景，看风景人在楼上看你。明月装饰了你的窗子，你装饰了别人的梦。

                          一个人心理衰老有两个标志：抱怨和后悔越来越多；把希望寄托在下一代。

                          我总觉得我会孤独终老，不期然有人穿过人群朝我走来。

                          日落尤其温柔，人间皆是浪漫。

                          从此唯行乐，闲愁奈我何。

                          且将诗酒瞒人眼，出入红尘过几冬。

                          所念皆星河，星河中是你！ 愿星河徜徉，一路有光。

                          孤独的人有他们自己的泥沼。

                          也许爱不是热情，也不是怀念，不过是岁月，年深月久成了生活的一部份。

                          # 但将冷眼看世界

                          你眼里有星星，有我

                          胆怯并不一定是弱点，正视自己害怕的东西，反而能成为我们成长的动力

                          我很悲伤，却假装很快乐，因为我知道你们喜欢我开心的样子

                          他朝若是同淋雪，此生也算共白头

                          但将冷眼看世界，看你横行到几时

                          若能避开猛烈的欢喜，自然不会有悲痛来袭

                          知行合一，经世致用

                          只怕功名不到头

                          从此以后，只有死别，没有生离

                          自今而后，愿意此生，扶佑君旁

                          待到春暖花开时，我要去大街上，见人问一句，你好

                          谦谦君子，当定而不疑

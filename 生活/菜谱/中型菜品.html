                                            菜品稍显复杂，肉食动物的最爱

                                            # 酱烧肥肠

                                            料酒焯水，姜葱炝锅，下洋葱，青红椒炒香下大肠

                                            锅边加醋，酱油生抽，盐

                                            出锅前加香油，蒜末

                                            # 把子肉

                                            1、传说中刘关张桃园结义拜把子的菜品

                                            2、精品五花肉烧皮洗净，加入葱姜料酒焯水；切成片，不要太厚

                                            3、菜籽油爆香，加入白糖炒至焦糖色；加入五花肉翻炒上色

                                            4、加入香料、葱姜、蚝油生抽炒香

                                            5、啤酒与足量水转入砂锅，大火烧开，转小火慢炖

                                            6、40分钟到1个半小时（根据自己的口味），调味起锅

                                            # 黄豆闷猪蹄

                                            1、猪蹄烧毛，去除皮腥味；黄豆提前泡水

                                            2、姜片料酒焯水

                                            3、菜籽油爆香，加入白糖炒至焦糖色

                                            4、两片八角，几只辣椒，盐，蚝油调味

                                            5、啤酒或水转入炖锅，蹄筋键，可以提前5~10分钟出锅

                                            6、转入锅中收汁调味

                                            # 小龙虾虾尾

                                            1、虾尾洗净，菜籽油爆炒盛出

                                            2、火锅底料炒化，大葱，姜蒜爆香，加入小龙虾尾爆炒

                                            3、加盐，蚝油，啤酒煮15分钟左右，加入生抽鸡精调味

                                            4、黄瓜打底，起锅装盘

                                            # 蒜蓉小龙虾

                                            1、小龙虾洗净，少油爆香盛出

                                            2、起锅少油，下入姜蒜末爆香，加入小龙虾

                                            3、一罐啤酒焖10分钟

                                            # 红烧狮子头

                                            1、肉剁碎、面粉、莲藕、胡萝卜丁，打入一个鸡蛋拌匀调味，中间少量多次加入葱姜水

                                            2、摔打上劲

                                            3、丸子下油锅炸

                                            # 焖饭

                                            1、 排骨，腊肠切片，黄瓜切丝，脆山药，香菇，胡萝卜切丁

                                            2、 香菇焯水

                                            3、 烧油炒糖色，排骨煎焦下入腊肠煎一会，加入胡萝卜、香菇爆炒后加水

                                            4、 转入电饭煲中，注意，水要比煮饭少蛮多的

                                            5、 快熟后加入山药（可以不加，痒手）

                                            6、 吃时撒上黄瓜丝

                                            注意点:

                                            1、焖饭放的水比正常煮饭要少

                                            # 煎泥鳅

                                            1、 泥鳅去除内脏，里面的东西都去掉

                                            2、 加油大火烧开煎后转小火慢慢煎，多煎一会，煎焦

                                            3、 加入葱姜，辣椒（辣的那种）

                                            4、 继续煎，并调味，起锅

                                            注意点：

                                            1、一定要尽可能的煎焦

                                            2、泥鳅如果太大要去除头和内脏

                                            # 猪肚鸡

                                            1、 猪肚洗净（使用面粉盐清洗多次）

                                            2、 猪肚焯水（多炒一会）后切片、鸡肉焯水

                                            3、 烧水菜籽油爆炒一下鸡肉和猪肚

                                            4、 加水后倒掉（去油），重新加水转入砂锅中炖（如果鸡是外面买的，鸡晚点下），姜片

                                            5、 灵魂胡椒粉、盐、生抽调味

                                            # 土豆炖牛腩

                                            1、牛腩焯水

                                            2、番茄烫一下去皮切小块

                                            3、土豆切块洗去淀粉

                                            4、起锅烧油，炒个糖色，加入牛肉，姜片翻炒均匀调味

                                            5、加水，土豆，番茄，刚没过就行，转压力锅炖20分钟

                                            6、倒出大火收汁，补味

                                            # 泡椒牛蛙

                                            1、 香菇、花菜焯水（香菇多炒一会）、青红椒切片备用

                                            2、 牛蛙剁块，用葱姜料酒腌制10分钟

                                            3、 牛蛙沥干水分，下入菜籽油爆香后捞出（煎焦）

                                            4、 余油加入火锅底料炒化（去掉火锅底料的油，肥仔不爱吃）加入土豆片，火腿，大葱、姜蒜翻炒，然后加入牛蛙，翻炒，加入耗油生抽老抽调味

                                            5、 加足量水与泡椒泡菜烧开转小火锅开吃，边下蔬菜

                                            # 红烧牛蛙

                                            牛蛙，辣椒（辣的那种），甜面椒

                                            牛蛙少许油炒一道水，下油炸焦，下辣椒，调味

                                            # 江苏肉蟹煲

                                            1、螃蟹清除腮和胃，拨壳分两块，粘上淀粉；虾清理干净；鸡爪清理指甲一分为2；

                                            2、莴笋、花菜等蔬菜焯水

                                            3、鸡爪焯水后然后用卤料包煮烂

                                            4、准备酱料生抽，耗油等，加入甜面酱加入水调成酱汁

                                            5、热锅下菜籽油，下入沾好淀粉螃蟹炸，注意要先煎至定型才能翻动，螃蟹煎好后起锅；

                                            6、下入大虾煎香起锅；

                                            7、锅中加油，炒糖色，下入鸡爪炒上色，加入老抽耗油炒香，加水及调好的酱汁，大火烧开然后下入炸好的螃蟹和大虾

                                            8、大火烧开闷一会，加入蔬菜转小火焖3分钟盛出，放入火锅中边吃边小火加热，香

                                            注意点:

                                            1、甜面酱可以多放一点，灵魂

                                            2、鸡爪汤一定要煮出胶质感

                                            # 甲鱼红烧肉

                                            1、 甲鱼五花肉分开焯水

                                            2、 爱吃的蔬菜焯水备用（莴笋、花菜等）

                                            3、 五花肉小火慢慢煎出油起锅，余油炒糖色后下午五花肉上色，加半瓶啤酒转砂锅小火慢慢焖肉（肉多焖个20分钟）；

                                            4、 另起锅菜籽油爆炒一下甲鱼，下入大葱、姜蒜，老抽翻炒上色，然后加入剩下的啤酒与水，烧开后转入五花肉一起炖

                                            5、 加入干辣椒；

                                            6、 炖到40分钟左右加入耗油生抽胡椒粉甜面酱调味。

                                            7、 转小火锅边吃边加入蔬菜

                                            # 清蒸鲈鱼

                                            1、鲈鱼洗净，把红色的肉和头腮洗净

                                            2、开背葱姜水腌制10分钟

                                            3、放两片姜，水开上汽蒸8分钟

                                            4、挑出姜片，铺上三丝，浇上热油（肥仔不喜欢油腻，不要浇油），淋上蒸鱼豉鱼

                                            # 红烧鳊鱼

                                            1、 鲳鱼，大葱丝，甜椒块

                                            2、 鱼清洗干净，划刀，用盐和料酒腌制10分钟左右；

                                            3、 锅中加油，加入少许盐，煎鱼至两面焦黄（倒出剩油，肥仔不喜欢油腻），加入姜蒜煸炒一会，然后加入一碗水烧开煮10分钟；

                                            4、 鱼起锅留汤汁，加入大葱甜椒、辣椒面，勾芡（肥仔不喜欢勾芡）调味，浇到鱼上。

                                            # 苏式红烧肉

                                            1、 精品五花肉，切块，简单焯水

                                            2、 热锅下少量油，将五花肉煎一会（少煎一会，煎久了瘦肉会柴，肥仔不喜欢），沥干油盛出，锅中的剩油用来炒糖色，下入肉上色，加入2大罐啤酒，加入卤料包（肥仔不喜欢香料味）大火烧开转小火焖

                                            3、 可以加入板栗、鹌鹑蛋等

                                            # 麻辣香锅

                                            1、瘦肉，鱼虾，鸡等任意 肉食切薄片

                                            2、菌菇类，莴笋，豆皮，土豆，芋头，海带，青菜，木耳，面食等任意 分批烫熟

                                            3、姜蒜切片

                                            4、放油烧热小火下火锅底料耗油，倒入蒜末姜片大料辣椒等，

                                            5、下入肉食炒熟加小半碗水烧开，倒入素菜少许盐快熟炒匀起锅。

                                            # 泥鳅炖豆腐

                                            1、 泥鳅剪头去内脏，下油放盐煎焦起锅

                                            2、 加油下姜蒜爆香，下泥鳅加水和豆腐炖烂，调味

                                            # 鱼冻

                                            1、 鱼腌制去腥

                                            2、 煎鱼块

                                            3、 加水煮，不要太多，太多不好冻

                                            # 剁椒鱼头

                                            1、 胖头鱼鱼头，腌制5分钟

                                            2、 甜椒、辣椒剁碎

                                            3、 盘子底下加姜、大蒜、加几片咸肉垫底（很关键）

                                            4、 加鱼头，辣椒均匀，加火腿豆豉

                                            5、 上锅蒸8分钟

                                            6、 加入生抽、蒸鱼豉油

                                            7、 热锅烧鱼、浇到鱼头上

                                    # 韭菜香干

                                    先下香干

                                    # 泡椒鸡胗

                                    # 酸辣土豆丝

                                    土豆丝洗净淀粉

                                    # 辣子鸡

                                    不要用老母鸡，用小鸡仔

                                    1、鸡肉剁小块，焯水

                                    2、鸡块炸干

                                    3、小块火锅底料化开，加入鸡块，各种辣椒翻炒均匀，调味

                                    4、撒入芝麻起锅

                                    # 可乐鸡翅

                                    # 小炒牛肉

                                    1、牛肉切片，一个鸡蛋少于盐，胡椒粉调味腌制10分钟

                                    2、起锅少油加入打算，芹菜，牛肉下锅爆炒，少许生抽调味迅速起锅

                                    3、不要太久，不然牛肉太老

                                    # 红烧鸡爪

                                    1、鸡爪一分两半

                                    2、炒糖色爆炒鸡爪

                                    3、加水，盐，蚝油转压力锅炖烂

                                    4、收汁调味

                                    # 藜蒿炒腊肉

                                    1、 藜蒿切小段，腊肉切薄片

                                    2、 腊肉小火煎一下，不要煎太焦（根据肥瘦程度掌握），熬出油，下入大蒜和藜蒿

                                    3、 调味起锅

                                    # 火腿煎蛋

                                    1、 火腿切片，鸡蛋加盐拌匀

                                    2、 热锅加油，加入火腿煎微微焦

                                    3、 倒入蛋液煎定型后反面

                                    # 黄瓜火腿

                                    1、 火腿、黄瓜切片

                                    2、 火腿煎一会，下入黄瓜爆炒

                                    3、 调味起锅

                                    # 干烧土豆片

                                    1、土豆切薄片，用水泡去淀粉； 熏肉切薄泡水去盐（或者使用新鲜肉片）

                                    2、多油，下入土豆片炸微微焦黄

                                    3、火锅底料炒化，加入姜蒜，倒入土豆片、肉片、耗油、辣椒面（酌情放盐）迅速翻炒

                                    4、锅边淋入少量水炒出锅气

                                    5、加入芝麻翻炒片刻起锅

                                    # 凉拌秋葵

                                    1、 秋葵分两半后焯水

                                    2、 准备朝天椒剁碎，生抽等做一个凉拌料

                                    3、 拌匀开吃

                                    # 油炸包菜

                                    1、 秋葵分两半后焯水

                                    2、 准备朝天椒剁碎，生抽等做一个凉拌料

                                    3、 拌匀开吃

                                    # 红枣蒸南瓜

                                    1、 老南瓜切大片、红枣对半分开

                                    2、 蒸5分钟

                                    # 小炒肉

                                    1、 选带劲的肉，螺丝椒（辣的那种），（可看喜好放豆豉）

                                    2、 下油煎肉至焦，倒入辣椒炒。

                                    3、 加盐，料酒（或者江小白），耗油，生抽起锅。

                                    # 粉蒸肉

                                    1、 葛粉调成糊糊，肉片均匀裹上

                                    2、 上蒸锅蒸
